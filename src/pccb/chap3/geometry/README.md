## Coneology POJ 2932
[2932 \-\- Coneology](http://poj.org/problem?id=2932)

* There are in total N cones (1 ≤ N ≤ 40000)
* Cone i has radius and height equal to Ri, i = 1 … N
* A cone is a unfilled circle
* No cone touches another cone
