## Overview


## lower bound
* $n$,
* $a_{0} \le \cdots \le a_{n-1}$,
* $k$,

Find the minimum $i$ satisfy that $a_{i} \ge k$.
If it does not exists, print $n$.

Constraints

* $1 \le n \le 10^{6}$,
* $0 \le a_{0} \le a_{1} \le \cdots \le a_{n-1}<10^{9}$,
* $0 \le k \le 10^{9}$,
