## Problem
* $n$,
    * the number of item types
* $(a_{i})_{i=1}^{n}$,
    * the number of $i$-th item
* $m$,
    * the number of maximum partitions
* $M$,
    * a divisor

Find that the number of $m$ combinations from $n$ types of items.
The answer is the number divided by $M$.

## Example


## Restriction

* $1 \le n \le 1000$,
* $1 \le m \le 1000$,
* $1 \le a_{i} \le 1000$,
* $2 \le M \le 10000$,
